[ PROJECT_01 ]

Pulse — Real-Time Analytics Dashboard

React Node.js WebSockets PostgreSQL

A live operations dashboard that streams event data from microservices and renders it as interactive charts. Built to help small teams spot anomalies before they become incidents, without needing a dedicated observability stack.

  • Sub-second updates via a WebSocket event bus handling ~5k msgs/sec
  • Custom charting layer built on top of D3 for full control over rendering
  • Role-based dashboards so engineers, support, and leadership each see relevant views
  • Alerting rules engine with configurable thresholds and notification channels

Lessons learned centered on backpressure handling and making sure the UI stayed responsive even when the data firehose spiked during incidents.

[ PROJECT_02 ]

Routine — Habit & Focus Tracker

TypeScript Next.js SQLite PWA

A minimalist, offline-first habit and focus tracker designed for people who find most productivity apps too noisy. Everything works without an account, and data stays on-device by default.

  • Installable PWA with full offline support and local-first storage
  • Streak tracking, daily focus timer, and weekly review prompts
  • Optional end-to-end encrypted sync between a user's own devices
  • Keyboard-first interaction model for rapid daily logging

The biggest design challenge was making sync conflict resolution invisible to the user while still being reliable across flaky connections.

[ PROJECT_03 ]

Terraform Landing Zone Toolkit

Terraform AWS Python CI/CD

A reusable set of Terraform modules and pipelines for spinning up secure, multi-account AWS environments in under an hour, with sane defaults that follow the AWS Well-Architected Framework.

  • Modular VPC, IAM, logging, and networking baselines as versioned modules
  • Automated drift detection with scheduled plan-only pipeline runs
  • Policy-as-code guardrails enforced via pre-merge checks
  • Documentation generated automatically from module variables and outputs

Used internally to cut new-environment setup time from roughly two weeks of manual work down to a single guided pipeline run.
